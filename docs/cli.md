# 🚀 CLI SDK - ModelOS

Use the CLI to run, log, and test agents programmatically.

## 🧪 Run a prompt

```bash
modelos run --model gpt-4 --prompt "Explain entropy like I'm 5"
```

## 📜 Log to MCP

```bash
modelos log --input "What is AGI?" --model mistral
```

## 📦 Generate MCP file

```bash
modelos export --out my_workflow.mcp.json
```
